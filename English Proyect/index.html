HTML and CSS

HTML

HTML is the language we use to create web pages.

Example: With HTML, we can say ‘here goes a title’ or ‘here goes an image’.

CSS

CSS is the language we use to decorate our web pages.

Example: With CSS, we can say ‘the title should be red’ or ‘the image should be big’.
